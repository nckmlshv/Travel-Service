# База данных сервиса Бронирования

### Таблица Bookings
Содержит данные о состоянии распределённой бизнес-операции (Saga) и фиксирует этапы выполнения бронирования во внешних системах.

| Имя атрибута | Тип данных | Ограничения | Описание |
|---|---|---|---|
| `id` | `UUID` | `PK`, `NOT NULL` | Уникальный идентификатор бронирования |
| `trip_id` | `UUID` | `NOT NULL` | Идентификатор связанной поездки |
| `user_id` | `UUID` | `NOT NULL` | Идентификатор заказчика |
| `saga_status` | `VARCHAR(50)` | `NOT NULL`, `CHECK (saga_status IN ('PENDING', 'CONFIRMED', 'FAILED', 'COMPENSATING'))` | Статус сессии бронирования |
| `total_price` | `DECIMAL(12,2)` | `NOT NULL`, `DEFAULT 0` | Итоговая стоимость бронирования |
| `created_at` | `TIMESTAMPTZ` | `NOT NULL` | Время инициации бронирования |

## Таблица Booking_items
Содержит позиции, входящие в одно бронирование, например авиабилеты, номера в отеле или трансферы.

| Имя атрибута | Тип данных | Ограничения | Описание |
|---|---|---|---|
| `id` | `UUID` | `PK`, `NOT NULL` | Уникальный идентификатор позиции бронирования |
| `booking_id` | `UUID` | `NOT NULL` | Ссылка на родительское бронирование |
| `provider_id` | `VARCHAR(100)` | `NOT NULL` | Идентификатор внешнего сервиса для интеграционного шлюза |
| `type` | `VARCHAR(50)` | `NOT NULL` | Тип услуги: `FLIGHT`, `HOTEL`, `TRANSFER` и т.д. |
| `provider_status` | `VARCHAR(50)` | `NOT NULL` | Технический статус обработки у внешнего провайдера |
