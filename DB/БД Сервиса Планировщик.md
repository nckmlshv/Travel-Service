# База данных сервиса Планировщик

## Таблица Trips
Хранит актуальное состояние планов поездок. Используется для отображения данных пользователю и проверки версий при синхронизации.

| Имя атрибута | Тип данных | Constraint | Описание |
|--|--|--|--|
| `id` | `UUID` | `PK, NOT NULL` | ID поездки |
| `user_id` | `UUID` | `NOT NULL, INDEX` | Организатор поездки |
| `title` | `VARCHAR(255)` | `NOT NULL` | Название поездки |
| `plan_data` | `JSONB` | `NOT NULL` | Данные о поездке в бинарном формате: маршрут, отели, билеты |
| `version_vector` | `JSONB` | `NOT NULL` | Текущий вектор для разрешения конфликтов (счетчик изменений по клиентам|
| `status` | `VARCHAR(50)` | `NOT NULL CHECK(STATUS IN ('ACTIVE', 'ARCHIVED','DELETED'))` | Статус поездки |
| `updated_at` | `TIMESTAMPTZ` | `NOT NULL` | Время последнего успешного изменения |
